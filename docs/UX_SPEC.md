# UX Spec (Residential-First)

## Design Goals
- Calm, readable, low-friction interaction for home users
- Large targets and simple language
- No hidden navigation
- Always-visible Help and Home recovery paths

## Top-Level Navigation
- Home tiles for primary tasks only
- Sticky bottom nav on all screens:
  - Home
  - Back
  - Speak
  - Help

## Copy Style
Use plain, friendly labels:
- Open Email
- See Photos
- Go Home
- Get Help

Avoid technical phrasing and admin jargon.

## Safety UX Patterns
- Confirm before send/delete/risky web actions
- Show visible caution/blocked banners where needed
- Offer support escalation immediately on uncertain actions

## Residential Admin UX
- Single local settings area
- PIN-locked admin entry
- Settings focused on:
  - Reminders
  - Contacts
  - Favorites
  - Safety guardrails
  - Support visibility
